## Site SeCoT 2025
Aplicação do Projeto Hanna, feita em parceria com as entidades HackoonSpace e Maritacas GameDev, para a Semana da Computação e Tecnologia 2025, da UFSCar Sorocaba.

- [Saiba mais sobre a Hanna](https://projetohanna.com)
- [Saiba mais sobre a SeCoT](https://secot.com.br)
- [Saiba mais sobre o HackoonSpace](https://hackoonspace.com)
- [Saiba mais sobre a Maritacas GameDev](https://maritacasgamedev.itch.io/)

### Instalação
Este projeto usa [yarn](https://yarnpkg.com/) como principal gerenciador de pacotes.

Para instalar as dependências, use:
```
yarn install
````

### Execução
Para rodar localmente (script de ambiente de desenvolvimento):

```bash
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador.
